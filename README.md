# Leave Tracking System

A console-based Leave Tracking System built using Java to manage employee leave requests efficiently.

## Features

### Employee Features
- Apply for leave
- Check leave balance
- View leave request status
- Cancel pending leave requests

### Manager/Admin Features
- View all leave requests
- Approve leave requests
- Reject leave requests
- Track employee leave balance

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collections
- LocalDate API
- Scanner (Console Input)

## Project Structure

```text
LeaveTrackingSystem/
│── Employee.java
│── LeaveRequest.java
│── LeaveType.java
│── LeaveManagementSystem.java
│── Main.java
```

## OOP Concepts Used

- Classes and Objects
- Encapsulation
- Inheritance (extendable)
- Polymorphism (future-ready)
- Abstraction

## How to Run

### Compile:

```bash
javac *.java
```

### Run:

```bash
java Main
```

## Example Workflow

1. Add employees
2. Apply leave
3. View requests
4. Approve or reject requests
5. Cancel pending requests
